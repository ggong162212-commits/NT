# 🌴 나트랑 여행일정 — 7월 4일 (토)

선고은 · 고지은 나트랑 여행 일정표 (모바일 웹페이지)

`index.html` 파일 하나로 동작하며, 외부 의존성은 한글 폰트(Pretendard CDN)뿐입니다.
인터넷이 없거나 폰트가 안 불러와져도 시스템 폰트로 정상 표시됩니다.

## 📁 파일

- `index.html` — 일정표 본체 (이 파일만 있으면 됩니다)
- `README.md` — 설명 (선택)

## 🚀 GitHub Pages로 올리는 법

1. GitHub에서 새 저장소(repository)를 만든다. (Public 권장)
2. `index.html` 파일을 저장소에 올린다.
   - 웹에서: **Add file → Upload files** → `index.html` 끌어다 놓고 **Commit**
3. 저장소 상단 **Settings → Pages** 로 이동.
4. **Build and deployment → Source** 를 **Deploy from a branch** 로 두고,
   Branch를 **main / (root)** 로 선택 후 **Save**.
5. 잠시 뒤 페이지 상단에 주소가 나온다:
   `https://<내아이디>.github.io/<저장소이름>/`
   → 폰으로 이 주소를 열면 끝!

## 📱 사용 팁

- 일정 카드 오른쪽 **체크박스**를 누르면 그 일정이 접힙니다. (완료 표시, 새로고침해도 유지)
- 상단 **호텔 정보 / 아나만다라 정보 / 여행 주의사항** 탭을 누르면 펼쳐집니다.
- 📍 지도 버튼은 구글맵 새 탭으로 열립니다.
- 현재 시각에 해당하는 일정은 자동으로 강조되고, 아래 **"⏰ 지금 일정으로"** 버튼으로 이동합니다.
